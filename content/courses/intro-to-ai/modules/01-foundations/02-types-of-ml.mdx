# Types of Machine Learning

Machine learning is the backbone of modern AI. At its core, ML is about building systems that learn from data. But not all learning is the same — there are three fundamental paradigms.

## 1. Supervised Learning

In supervised learning, the model learns from **labeled examples**. You provide input-output pairs, and the model learns to map inputs to outputs.

**Examples:**
- Email spam detection (input: email text, output: spam/not spam)
- House price prediction (input: features like size and location, output: price)
- Image classification (input: image pixels, output: category label)

```python
# Simple supervised learning example
from sklearn.linear_model import LinearRegression

# Training data: square footage → price
X = [[1000], [1500], [2000], [2500]]
y = [200000, 300000, 400000, 500000]

model = LinearRegression()
model.fit(X, y)

# Predict price for a 1750 sq ft house
prediction = model.predict([[1750]])
print(f"Predicted price: ${prediction[0]:,.0f}")
```

<Callout type="tip" title="Remember">
Supervised learning requires labeled data. The quality and quantity of your labels directly impacts model performance.
</Callout>

## 2. Unsupervised Learning

In unsupervised learning, the model finds **patterns in unlabeled data**. There are no correct answers provided — the model discovers structure on its own.

**Common techniques:**
- **Clustering**: Grouping similar items (e.g., customer segmentation)
- **Dimensionality reduction**: Simplifying data while preserving patterns
- **Anomaly detection**: Finding unusual data points

```python
# Clustering example
from sklearn.cluster import KMeans

# Customer data: [annual_income, spending_score]
customers = [[15, 39], [16, 81], [17, 6],
             [85, 75], [90, 88], [87, 63]]

kmeans = KMeans(n_clusters=2)
kmeans.fit(customers)

print(f"Cluster centers: {kmeans.cluster_centers_}")
print(f"Labels: {kmeans.labels_}")
```

## 3. Reinforcement Learning

In reinforcement learning, an **agent** learns by interacting with an **environment**. It receives rewards or penalties based on its actions and learns to maximize long-term reward.

**Examples:**
- Game-playing AI (AlphaGo, Atari games)
- Robotics (learning to walk, grasp objects)
- Recommendation systems

The key components:
- **Agent**: The learner/decision maker
- **Environment**: What the agent interacts with
- **State**: Current situation
- **Action**: What the agent can do
- **Reward**: Feedback signal

<Callout type="info" title="Fun Fact">
DeepMind's AlphaGo, which defeated the world champion in Go, learned through a combination of supervised learning (studying human games) and reinforcement learning (playing against itself millions of times).
</Callout>

## Choosing the Right Approach

| Paradigm | You Have | Goal | Example |
|----------|----------|------|---------|
| Supervised | Labeled data | Predict outcomes | Spam detection |
| Unsupervised | Unlabeled data | Find patterns | Customer segments |
| Reinforcement | Environment + rewards | Optimize decisions | Game AI |
