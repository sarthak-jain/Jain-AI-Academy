# Setting Up Your Python Environment

Before we can build ML models, we need the right tools. This lesson walks you through setting up a complete Python environment for machine learning.

## Prerequisites

You'll need:
- A computer running Windows, macOS, or Linux
- An internet connection
- About 2GB of free disk space

## Step 1: Install Python

We recommend **Python 3.10+**. Download it from [python.org](https://python.org) or use a package manager:

```bash
# macOS (using Homebrew)
brew install python@3.11

# Ubuntu/Debian
sudo apt update && sudo apt install python3.11 python3.11-venv

# Windows: download from python.org and run the installer
# Make sure to check "Add Python to PATH"
```

Verify your installation:

```bash
python3 --version
# Python 3.11.x
```

## Step 2: Create a Virtual Environment

Virtual environments keep your project dependencies isolated:

```bash
# Create a project folder
mkdir ai-academy-lab && cd ai-academy-lab

# Create a virtual environment
python3 -m venv venv

# Activate it
# macOS/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate
```

<Callout type="tip" title="Pro Tip">
Always use virtual environments for ML projects. Different projects often need different versions of the same library.
</Callout>

## Step 3: Install Core Libraries

```bash
pip install numpy pandas scikit-learn matplotlib jupyter
```

Here's what each library does:

| Library | Purpose |
|---------|---------|
| **NumPy** | Numerical computing, arrays |
| **Pandas** | Data manipulation, DataFrames |
| **scikit-learn** | Machine learning algorithms |
| **Matplotlib** | Data visualization |
| **Jupyter** | Interactive notebooks |

## Step 4: Verify Everything Works

Create a quick test script:

```python
import numpy as np
import pandas as pd
import sklearn
import matplotlib

print(f"NumPy: {np.__version__}")
print(f"Pandas: {pd.__version__}")
print(f"scikit-learn: {sklearn.__version__}")
print(f"Matplotlib: {matplotlib.__version__}")
print("\nAll libraries installed successfully!")
```

Run it:

```bash
python test_setup.py
```

## Step 5: Launch Jupyter

For the hands-on exercises, we'll use Jupyter notebooks:

```bash
jupyter notebook
```

This opens a browser window where you can create and run interactive notebooks.

<Callout type="info" title="Alternative">
If you prefer, you can use **Google Colab** instead of a local setup. It's free and comes with all libraries pre-installed. Visit [colab.research.google.com](https://colab.research.google.com).
</Callout>

## You're Ready!

With your environment set up, you're ready to build your first ML model in the next lesson.
